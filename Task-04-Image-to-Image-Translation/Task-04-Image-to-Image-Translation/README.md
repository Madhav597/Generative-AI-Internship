# Task 04 - Image-to-Image Translation with cGAN (Pix2Pix)

This project demonstrates a simple implementation of image-to-image translation using a conditional GAN (cGAN) architecture inspired by Pix2Pix.

## Objective
To transform one image into another using a deep learning model.

## Technologies Used
- Python
- PyTorch
- GAN (Generative Adversarial Network)

## Steps
1. Load input image
2. Pass image through Generator network
3. Generate translated image
4. Save the output image

## Output
The generated image is saved as `generated_output.png`.
