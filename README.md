# Generative AI Internship – Prodigy Infotech

This repository contains all tasks completed during my **Generative AI Internship at Prodigy Infotech**.

The internship focuses on implementing different **Generative AI models**, including transformers, Markov chains, GANs, and neural style transfer.

---

## 🚀 Internship Tasks

### Task 01 – Text Generation with GPT-2
Generate coherent text using the GPT-2 transformer model.

📂 Folder: `Task-01-GPT2-Text-Generation`

---

### Task 02 – Image Generation
Generate images using generative AI techniques.

📂 Folder: `Task-02-Image-Generation`

---

### Task 03 – Text Generation with Markov Chains
Implemented a statistical model that generates text based on probability transitions between words.

📂 Folder: `Task-03-Markov-Text-Generation`

---

### Task 04 – Image-to-Image Translation with cGAN (Pix2Pix)
Implemented an image-to-image translation model using a **Conditional GAN (Pix2Pix)** architecture.

📂 Folder: `Task-04-pix2pix-Image-Translation`

---

### Task 05 – Neural Style Transfer
Applied the artistic style of one image to another image using neural networks.

📂 Folder: `Task-05-Neural-Style-Transfer`
