# Task 01 - Text Generation with GPT-2

This project demonstrates text generation using the GPT-2 transformer model.

## Objective
Train or use a transformer model to generate coherent and contextually relevant text based on a prompt.

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## How It Works
1. Load GPT-2 tokenizer and model
2. Provide a prompt
3. Generate text using the transformer model
