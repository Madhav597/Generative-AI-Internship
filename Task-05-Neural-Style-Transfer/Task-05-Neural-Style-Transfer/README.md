# Task 05 - Neural Style Transfer

This project applies the artistic style of one image to another image using Neural Style Transfer.

## Objective
To combine the **content of one image** with the **style of another image**.

## Technologies Used
- Python
- PyTorch
- VGG19 Model
- Neural Networks

## Steps
1 Load content image  
2 Load style image  
3 Extract features using VGG19  
4 Compute style and content loss  
5 Generate stylized output image  

## Output
The stylized image is saved as:

stylized_output.png
