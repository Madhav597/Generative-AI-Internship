# Task 03 - Text Generation using Markov Chains

This project implements a simple text generation algorithm using Markov Chains.

## Objective
To create a statistical model that predicts the next word based on the previous word.

## How it Works
1. Input text is split into words.
2. A dictionary (Markov Chain) stores word transitions.
3. The program randomly selects the next word based on probability.

- Python
- Random Library
