# Task 04 - Image-to-Image Translation using Pix2Pix (cGAN)

## Overview
This project implements an Image-to-Image Translation model using a Conditional Generative Adversarial Network (cGAN) inspired by the Pix2Pix architecture.

The model learns to transform an input image into another image representation.

## Technologies Used
- Python
- PyTorch
- GAN (Generative Adversarial Network)

## Project Files

pix2pix_model.py → Generator architecture  
generate_image.py → Script to generate images  
requirements.txt → Dependencies  

## How to Run

1 Install dependencies

pip install -r requirements.txt

2 Add input image

Place image named `input.jpg` in the folder.

3 Run

python generate_image.py

## Output

The generated translated image will be saved as:

output.png
